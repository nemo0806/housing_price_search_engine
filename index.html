Search Engine

This is a simple search engine for housing in China.

Note 1: the searching bar for "name" and "location" can be left blank.

Note 2: searching for multiple keywords is supported in the searching bar of "name" (each keyword should be separated by semicolon (;)), and the retrieved contents with the most corresponding keywords will be listed first.

Note 3: please type the smallest area you would accept in the searching bar of "area"

Note 4: please type the highest price you can afford in the searching bar of "price"

Special Notice: this website is for searching different contents separately in different searching bars, if you want a vague search, please click here.
